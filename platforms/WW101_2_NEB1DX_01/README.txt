--------------------------------------------
WW101_2_NEB1DX - README
--------------------------------------------

This platform is for the combination of the NEB1DX and version 2.* of the
CY8CKIT-032 PSoC AFE Shield.

Provider    : Future Electronics & Cypress Semiconductor
Website     : http://cypress.com/wiced
Description : Murata 1DX mounted on the Nebula Future Electronics evaluation board with an STM32F429

Module
  Mfr     : Murata
  P/N     : 1DX
  MCU     : STM32F429ZIT
  WLAN    : BCM4343W Wi-Fi CoB
  Antenna : Diversity with two printed antennae (and in-line switched Murata MM8430 RF connectors)

EVB Features
  JLINK-JTAG debug interface
  USB-JTAG debug interface
  USB-serial UART interface
  Power supply : USB and/or external +5v
  Reset button
  Module current monitor
  Sensors/Peripherals
     - 2 x Buttons
     - 4 x LEDs
     - 1 x Thermistor
     - 1 x 8Mbit serial flash

Shield Features
  I2C interface to PSoC and OLED display
  Analog Front End for POT, Thermistor, Ambient Light Sensor, and Humidity Sensor
  Adjustable VDAC voltage output
  4X CapSense buttons and 1X proximity sensor
  2x Mechanical Buttons
  7x LEDs (2 for mechanical buttons, 4 for CapSense buttons 1 for CapSense proximity)
  U8G 128x64 OLED dispaly
